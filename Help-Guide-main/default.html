Analyze in Excel allows datasets from reports to be brought into Excel where they can be interacted with using PivotTables, charts, slicers, and other Excel features.

Why choose ‘Analyze in Excel’ over ‘Export Data’?

Analyze in Excel is different from exporting data directly from a visual:

Analyze in Excel: Export data
Data from entire report Data from single visual
Live data connection that can be refreshed as the data changes Snapshot of data from when it was accessed
Live data connection that is tied to the credentials of the user Flat Excel file that can be shared with anyone

Single tap in reports:

Single tap allows you to interact with a visual, button, or slicer with a single tap. When Single tap is disabled, a single tap will select the item and a second tap starts the interaction.

Docked report footer:

When docked, the report footer is fixed to the bottom of the page. When this option is disable, the footer will hide and appear based on actions taken in the report, such as scrolling.

Multi-select:

With Multi-select, each tap on a data point is recognized alongside any other selected data points. When Multi-select is disabled, only a single data point can be selected at one time. Many reports at Costco utilize filters that utilize multi-select. For example, if you wanted to select multiple periods in a given year, or select multiple locations at once. Power BI disables Multi-select at default; to get the most out of the reports it is recommended to enable this setting.

Data reader (accessibility):

Data reader will read visuals out loud. To activate, make sure ‘TalkBack’ is enabled, and tap ‘Show as a table’ on the visual you want spoken.

Require Biometric Authentication:

Biometric Authentication allows Power BI to use your device’s fingerprint scanner or facial recognition when using the app.
